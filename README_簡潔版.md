# 🐉 天堂2M BOSS重生追蹤器

## 🌟 功能特色
- 📱 **響應式設計** - 手機、平板、電腦完美適配
- 👥 **多人共享** - 所有用戶共享BOSS數據
- ⚡ **即時更新** - 修改立即同步給所有用戶
- 🎯 **61個真實BOSS** - 完整天堂2M BOSS列表

## 🚀 部署到雲端

### 1. 上傳到 GitHub
創建倉庫並上傳這3個檔案：
- `app.py` (主程式)
- `requirements.txt` (依賴)  
- `README.md` (說明)

### 2. 部署到 Streamlit Cloud
1. 前往：https://share.streamlit.io
2. 用 GitHub 登入
3. 選擇 "New app"
4. 選擇您的倉庫
5. Main file: `app.py`
6. 部署完成！

## 💡 使用說明

### 基本操作
1. **選擇BOSS** - 使用下拉選擇器
2. **記錄擊殺** - 點擊「記錄現在時間」
3. **自定義時間** - 手動輸入特定時間
4. **清除記錄** - 清除單個或所有BOSS記錄

### 時間格式
- `2025/08/11 16:30` (完整格式)
- `08/11 16:30` (簡化格式)
- 支援多種分隔符號

### 狀態指示
- ✅ **已重生** - 可以挑戰
- ⏳ **等待中** - 顯示剩餘時間
- ⚪ **未記錄** - 尚未記錄擊殺
